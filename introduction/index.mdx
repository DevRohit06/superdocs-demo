# Welcome to SuperDocs

SuperDocs is a powerful, flexible, and design-forward documentation generator built on top of Astro. It allows you to create stunning documentation sites with minimal configuration, while giving you the full power of Astro and MDX when you need it.

## Philosophy

SuperDocs was built with the belief that documentation should be:

<Steps>
  <Step title="Beautiful by default">
    <Icon name="lucide:sparkles" size={18} class="text-primary-500" /> You shouldn't need to be a designer to have good looking docs.
  </Step>
  <Step title="Fast">
    <Icon name="lucide:rocket" size={18} class="text-primary-500" /> Both for the end user and the developer.
  </Step>
  <Step title="Extensible">
    <Icon name="lucide:puzzle" size={18} class="text-primary-500" /> Easy things should be easy, hard things should be possible.
  </Step>
</Steps>
