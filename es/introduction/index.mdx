# Bienvenido a Lito

Lito es una alternativa de código abierto a Mintlify que te permite crear sitios de documentación hermosos a partir de archivos Markdown.

## Características Principales

- **Basado en Astro** - Rendimiento ultrarrápido con generación de sitios estáticos
- **MDX Incluido** - Escribe Markdown con componentes interactivos
- **Tema Hermoso** - Diseño profesional listo para usar
- **Búsqueda Integrada** - Búsqueda con Pagefind incluida
- **Documentación de API** - Páginas de referencia de API generadas automáticamente
- **i18n** - Soporte multilenguaje incorporado

## ¿Por qué Lito?

Lito te da el control total sobre tu sitio de documentación mientras proporciona una experiencia similar a Mintlify, pero de código abierto y autoalojado.
