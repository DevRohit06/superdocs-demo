## Overview

Retrieves all comments associated with a specific blog post.

## Path Parameters

<ParamTable type="path">
  <Param name="id" type="integer" required>
    The ID of the post to get comments for
  </Param>
</ParamTable>

## Response

Returns an array of comment objects.

<ResponseField name="id" type="integer">
  Unique identifier for the comment
</ResponseField>

<ResponseField name="postId" type="integer">
  ID of the post this comment belongs to
</ResponseField>

<ResponseField name="name" type="string">
  Title or subject of the comment
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the commenter
</ResponseField>

<ResponseField name="body" type="string">
  Content of the comment
</ResponseField>
